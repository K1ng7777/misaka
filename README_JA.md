このアプリケーションの目的は、iPhoneユーザーが脱獄せずにiOSをカスタマイズできるようにすることです。

現時点では、24 を超えるデフォルト リポジトリと 400 を超える調整が利用可能です。

*Misakaは無料で提供されています誰かがあなたにそれを販売しようとした場合、それを報告してください*

<br>

## サポートされているバージョン
| iOSバージョン | MDC | KFD |
| - | - | - |
| 15.0 - 15.1.1 | ✓ | - |
| 15.7.2 - 15.8.1 | - | ✓ |
| 16.0 - 16.1.2 | ✓ | - |
| 16.2 - 16.6.1 | - | ✓ |
| 17.0 & up | - | - |

17.0以上はPiPの微調整を使用できます
